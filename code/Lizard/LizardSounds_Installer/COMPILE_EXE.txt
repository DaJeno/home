HOW TO CREATE THE lizard_sounds.exe FILE
========================================

Since this package was created on Linux, the .exe file needs to be compiled on Windows.

OPTION 1: Compile the C code (Recommended)
------------------------------------------
1. Install a C compiler like MinGW-w64 or Visual Studio
2. Open Command Prompt on Windows
3. Navigate to this folder
4. Run: gcc lizard_sounds_launcher.c -o lizard_sounds.exe

OPTION 2: Use the Python script
-------------------------------
1. Install Python on Windows
2. Install pyinstaller: pip install pyinstaller
3. Run: pyinstaller --onefile --noconsole lizard_sounds_exe.py
4. Rename the resulting .exe to lizard_sounds.exe

OPTION 3: Simple batch file renamed
-----------------------------------
1. Rename lizard_sounds_run.bat to lizard_sounds.exe
2. Windows will run it as a batch file despite the .exe extension

OPTION 4: Use a Batch-to-EXE converter
--------------------------------------
Use tools like:
- Bat To Exe Converter
- Advanced BAT to EXE Converter
- QuickBFC (Quick Batch File Compiler)

The .exe should simply run: AutoHotkey.exe lizard_sounds.ahk
